Necesitamos organizar la información de las películas que nos gustan :movie_camera:, para eso vamos a crear un código HTML que contenga:

- un título `h1` con el nombre de una película.
- una imagen con el poster de la película.
- un `h2` que diga: `Elenco`
- un lista desordenada con 3 nombres del elenco. Los nombres deben ser links a la página de IMDb correspondiente (por ejemplo, [Anne Hathaway](https://www.imdb.com/name/nm0004266/?ref_=nv_sr_srsg_0))
- un título `h3` que diga: `Sinopsis`
- un párrafo `p` con la sinopsis de la película
- un `h3` que diga: `Géneros`
- un lista desordenada con 3 géneros de la película. Los nombres deben ser links a la página de IMDb correspondiente de la lista de películas de dicho género (por ejemplo, [Thriller](https://www.imdb.com/search/title/?genres=thriller&explore=title_type,genres&ref_=tt_ov_inf))

> Usando [IMDb](https://www.imdb.com/) como referencia, creá el código HTML que acabamos de describir.
